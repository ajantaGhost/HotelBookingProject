<h3 id="project1">Project 1: Hotel Booking Intelligence Pipeline </h3>
<h4>From Raw Data to Predictive Insights</h4>
<p>
  The Hotel Booking Intelligence Pipeline project focuses on analyzing hotel reservation data to optimize revenue, reduce booking cancellations, and understand customer behavior. The primary objective was to transform raw booking data into meaningful insights and predictive models that support data-driven decision-making in the hospitality industry.

An end-to-end analytics pipeline was implemented using Excel, Python. Excel was used for data cleaning, enrichment, and exploratory analysis. Python was applied for predictive modeling, feature engineering, and automation.

Key insights revealed that lead time, room type, booking channel, and customer ratings significantly impact revenue and cancellation behavior. Predictive models successfully classified booking status and repeted guest, helping identify high-risk cancellations and high-value bookings. Overall, the project demonstrates a complete real-world data analytics workflow from raw data ingestion to actionable and predictive intelligence.
</p>
